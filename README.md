How to Use the Tracker

Access: Click the link above on your phone, tablet, or desktop.

Daily Check-in: Each day, check off the key habits you completed (Hydration, Movement, Blueprint Adherence, Sleep).

Track Your Energy: Use the quick energy rating to understand your body's response over time.

Maintain the Chain: The goal is to keep your daily streak going for 21 consecutive days to establish a deep, sustainable habit loop.

This project is built as a lean, single-page web application to ensure fast load times and universal accessibility.

Technologies Used

HTML5: Core structure.

Tailwind CSS: For all styling, ensuring a clean, responsive, and mobile-first design.

Vanilla JavaScript: Handles all interactive logic and state management (tracking checkmarks).

Local Storage (Default): Used to persist the user's progress locally within their browser.
